# Web Testing Lab

This is a simple browser-based lab to test JavaScript payloads using URL parameters or manual input. It is designed for learning how client-side execution works.

## Features

* Execute JavaScript using the `?script=` parameter
* Hidden console panel for manual testing
* Clean and minimal UI
* Works easily with GitHub Pages

## Usage

### URL Execution

You can pass JavaScript directly in the URL:

```
?script=alert(1)
```


### Manual Testing

* Click on "Open Console"
* Enter your JavaScript payload
* Click "Execute"

## Important Notes

* This project is for learning and testing only
* It uses `eval()`, which is unsafe by design
* Do not use this in production or on real users

## What You Can Practice

* DOM manipulation
* Understanding how JavaScript executes in the browser
* Testing simple payload behavior
